# 🌡️ 黑体辐射模拟器

基于普朗克黑体辐射定律的交互式可视化工具

## 🎯 功能特点
- 📊 实时调节温度（500-50,000 K）
- 🌈 可见光区域彩色光谱显示
- 📈 显示峰值波长、总辐射功率、可见光比例
- 🔬 对比经典公式（瑞利-金斯、维恩）
- 🎨 动态黑体颜色模拟
- 🎬 动画演示功能

## 🚀 在线体验
[点击这里访问在线版本](https://your-app.streamlit.app)

## 💻 本地运行
1. 安装依赖：`pip install -r requirements.txt`
2. 运行程序：`streamlit run app.py`
3. 浏览器访问：`http://localhost:8501`

## 📚 理论基础
- 普朗克黑体辐射定律
- 维恩位移定律
- 斯特藩-玻尔兹曼定律
